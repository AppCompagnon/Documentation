# Conditions d'utilisation et d'information

### 1. Utilisation de vos données

Compagnon ne partage aucune donnée, vos données sont stockées localement dans votre appareil.

### 2. Termes d'utilisation des applications tiers

D'autres applications sont utilisées pour pouvoir garder Compagnon actif. Dans ce cas-là, le respect de leur termes est nécéssaire.

* **Codage** : GitHub
* **Documentation** : Gitbool

### 3. Utilisation pour des élèves

L'application est destinée aux élèves, Compagnon ne contient aucune image ou contenu choquant sur l'entièreté de sa plateforme. Dans le cas échéant, informez-nous en via le mail suivant : compagnon@mail.com.

### 4. Clause de non-dommage

Compagnon respecte les règles suivantes :

* Application sécurisée
* Données non partagées (local)
* Code opensource

Compagnon n'est pas responsable, comme noté dans l'intérieur de l'application lors du lancement :&#x20;

* D'une donnée (notes, cours, devoirs, etc.) éronné&#x20;
* D'un cours mal affiché (par exemple d'un cours annulé alors que non)

La lecture des conditions est obligatoire dès le premier lancement de l'application.

### 5. Entreprise

Contrairement à ce que certains peuvent le penser, Compagnon n'est **pas** une entreprise, ni une organisation (légalement). Compagnon ne récupère cependant **aucun** don.
