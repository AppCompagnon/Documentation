# Conditions d'utilisation

### 1. Utilisation de vos données

Compagnon ne partage aucune donnée, vos données sont stockées localement dans votre appareil.

### 2. Termes d'utilisation des applications tiers

D'autres applications sont utilisées pour pouvoir garder Compagnon actif. Dans ce cas-là, le respect de leur termes est nécéssaire.

* **Codage** : GitHub
* **Documentation** : Gitbool

### 3. Utilisation pour des élèves

L'application est destinée aux élèves, Compagnon ne contient aucune image ou contenu choquant sur l'entièreté de sa plateforme. Dans le cas échéant, informez-nous en via le mail suivant : compagnon@mail.com.
